## What is Templatingway?
Templatingway is the standard template repository for all NAUR repositories.

Use this section to describe what your project is or what it does.

## Getting started
To start developing Templatingway you need just a few things:

- [.NET 10 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
- [Aspire CLI](https://aspire.dev/get-started/install-cli/)
- [Docker Desktop](https://www.docker.com/products/docker-desktop/), [Podman Desktop](https://podman-desktop.io/), or [Rancher Desktop](https://rancherdesktop.io/)

## Run Templatingway
Once you have all the dependencies listed above simply execute `aspire run` to start Templatingway. You can access the following services:

- Aspire Dashboard: https://aspire.dev.localhost:8440
- Templatingway: https://templatingway.dev.localhost:8443

The Aspire dashboard may prompt for additional values in order to start Templatingway, filling these out will start the service.
