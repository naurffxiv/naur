# Contributing

Contributions are welcome and valued.

## Quick Start

1. **Setup**:

    ```bash
    make setup
    make install
    ```

2. **Verify**:

    ```bash
    make lint   # Check code style
    make check  # Check environment tools
    ```

For a full list of commands, run:

```bash
make help
```

## Troubleshooting

```bash
make troubleshoot
```

## Guidelines

- Self-review your code
- Test your changes locally
- Link a ticket in your PR on a new line (use `Fixes #123`, `Ticket #123`, or `Ticket NA`)
- Follow our [Code of Conduct](CODE_OF_CONDUCT.md)

## Questions?

Join our [Discord](https://discord.gg/naurffxiv) or open an issue!
